# Dan Koe 一日人生重啟網站 — 設計文件

**日期：** 2026-04-01  
**類型：** 個人反思日誌系統  
**部署：** 純靜態單頁 HTML，無需伺服器

---

## 概覽

一個讓使用者執行 Dan Koe「一日人生重啟」方法論的互動式日誌網站。使用者在一天內完成三個階段（早晨／白天／晚上）的反思題，資料儲存於瀏覽器 localStorage，可隨時回來繼續或檢視舊紀錄。

---

## 視覺風格

- **配色：** 暖色系手帳感，背景 `#ede8e0`，卡片 `#f0e9dc`，輸入框 `#e8e0d4`，主色 `#8b6914`（深金棕）
- **字體：** Georgia / 微軟正黑體，serif 系列
- **字級：** 主標題 52px、區塊標題 30px、問題文字 22px、輸入框 18px
- **無純白底色**，全站用米黃與棕灰色系，減少視覺刺激
- **語言：** 全繁體中文

## 資料儲存（localStorage）

### 儲存格式

```json
{
  "records": [
    {
      "date": "2026-04-01",
      "morning": {
        "reverse_vision": "...",
        "emotion_observation": "...",
        "positive_vision": "...",
        "identity": "..."
      },
      "daytime": {
        "self_audit": "...",
        "camera_metaphor": "..."
      },
      "evening": {
        "baseline": "...",
        "direction": "...",
        "goal_1year": "...",
        "goal_1month": "...",
        "goal_tomorrow": "..."
      },
      "completed": false
    }
  ]
}
```

### 行為
- **自動存草稿：** 每個輸入框 `input` 事件觸發，debounce 1 秒後存入 localStorage
- **載入：** 頁面開啟時讀取今日日期的紀錄，若有則填入各欄位
- **完成：** 點「完成今日重啟」後將 `completed` 設為 `true`，進度點全亮
- **匯出：** 點「匯出紀錄」後將今日紀錄序列化為 JSON，觸發瀏覽器下載

2. 關閉瀏覽器再開，當日填寫內容完整保留
3. 可成功匯出 JSON 檔案
4. 全站無純白底色，字體清晰易讀
5. 不需要安裝任何套件或啟動伺服器，直接開啟 `index.html` 即可使用
