## 3. 計算邏輯

### 3.1 凱利公式（kelly.py）

```
原始凱利% = (勝率 × 賠率 - 敗率) / 賠率
敗率 = 1 - 勝率

調整後凱利% = 原始凱利% × 週期共振乘數 × 費波那契入場乘數
全凱利倉位 = 初始資金 × 調整後凱利%
半凱利倉位 = 全凱利倉位 × 0.5
```

### 3.2 費波那契整合（fibonacci.py）

標準回檔位及對應乘數：

| 回檔位 | 入場乘數 | 風險報酬比（自動帶入賠率） |
|--------|---------|--------------------------|
| 23.6%  | 0.6     | 1.5                      |
| 38.2%  | 0.8     | 2.0                      |
| 50.0%  | 0.9     | 2.5                      |
| 61.8%  | 1.0     | 3.0                      |
| 78.6%  | 1.1     | 4.0                      |

- **入場乘數**：回檔越深代表支撐越強，乘數越大，凱利倉位相應放大
- **賠率自動計算**：從選定回檔位到下一阻力位距離，自動帶入風險報酬比，使用者也可手動覆蓋

### 3.3 四週期共振分數（cycles.py）

使用者輸入各週期當前位置（整數 -2 到 +2）：
- `-2`：深度收縮
- `0`：中性
- `+2`：強勁擴張

加權平均計算共振分數：

| 週期     | 週期長度  | 權重 |
|----------|---------|------|
| 基欽週期  | 3-4 年  | 15%  |
| 朱格拉週期| 7-11 年 | 25%  |
| 庫茲涅茨  | 15-25 年| 30%  |
| 康波週期  | 50-60 年| 30%  |

共振分數映射到乘數：
- 分數 -2.0 → 乘數 0.5（全收縮，大幅縮倉）
- 分數 0.0 → 乘數 1.0（中性，原始凱利）
- 分數 +2.0 → 乘數 1.5（全擴張，放大倉位）

### 3.4 蒙地卡羅模擬（montecarlo.py）

- 每次模擬：以輸入勝率進行 N 筆（預設 200 筆）二元交易，依調整後凱利倉位計算每筆資金變化
- 重複 1000 次，numpy 向量化執行，< 1 秒完成
- 輸出：
  - 1000 條資金曲線
  - 5% / 50% / 95% 分位線
  - 最大回撤分佈（均值、標準差、最差值）
  - 破產率（資金跌破初始資金 50% 的路徑比例）

## 4. 網頁介面（index.html）

單頁布局，分左右兩欄：

**左欄（輸入）：**
- 基本參數：初始資金、勝率（%）、費波那契回檔位選擇（下拉）
- 週期位置：基欽 / 朱格拉 / 庫茲涅茨 / 康波（各 -2 到 +2 滑桿）
- 送出按鈕

**右欄（結果）：**
- 原始凱利%
- 週期共振分數 + 乘數
- 費波那契入場乘數
- 調整後凱利%
- 建議全凱利倉位（$）
- 建議半凱利倉位（$）
- 下載 Excel 按鈕

## 5. Excel 輸出（excel_builder.py）

共 4 個工作表，使用 xlsxwriter：

| 工作表       | 內容                                              |
|------------|--------------------------------------------------|
| **總覽**     | 所有輸入參數 + 計算結果摘要表                        |
| **蒙地卡羅曲線** | 1000 條資金路徑（取樣 100 條顯示）+ 分位線折線圖   |
| **回撤分析** | 最大回撤直方圖 + 統計表（均值/標準差/最差值/破產率） |
| **週期共振** | 四週期輸入值對照表 + 調整前後凱利% 比較             |
